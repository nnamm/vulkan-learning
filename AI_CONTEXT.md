# Project Context

This repository is a Vulkan learning project based on a book and sample code.

## Coding Policy

- I am typing the book/sample code by hand for learning.
- Prefer readability and explicitness over minimizing includes.
- Even if the book/sample relies on transitive includes, add direct includes for types and functions used in each file.
- Do not aggressively refactor code away from the book structure unless I ask.
- If a symbol is undefined, first explain whether it is Vulkan standard, C++ standard, or project-defined.
- When suggesting fixes, prefer small changes that preserve the learning flow.

- Vulkan / C++
- CMake
