# Vulkan Learning

This repository contains my personal code-along implementation of the book, "[Vulkan実践入門 グラフィックスの基礎からレイトレーシング、メッシュシェーダーまで](https://gihyo.jp/book/2025/978-4-297-15257-4)". (Practical Introduction to Vulkan: From Graphics Fundamentals to Ray Tracing and Mesh Shaders.)

For development on Windows11, I primarily utillze [Zed](https://zed.dev/) and [CMake](https://cmake.org/), with Visual Studio 2026 serving as the compiler.

Throughout this project, I am studying modern development practices with CMake, using the provided Visual Studio 2026 sample programs as a reference for best practices.
